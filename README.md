# 仙劫 · 幸存（修仙幸存者）

修仙题材的《吸血鬼幸存者》范式 Roguelite 小游戏，面向微信小游戏。

- 操作极简：只控制移动（WASD / 虚拟摇杆），攻击全自动
- 单局短、随机构筑、标签羁绊（剑/刀/仙/魔/妖/体/雷/阵/木）
- 纯客户端单机，无服务端、无多人、无登录

> 由原"修仙搜打撤 PvPvE"原型转型而来。原始重度版本及其 Node 服务端备份在 `prototype-3d.backup/`。
> 旧版设计文档仍在 `docs/`，仅供世界观/美术参考，玩法以本 README 为准。

## 运行

```powershell
cd prototype-3d
npm install
npm run dev -- --host 127.0.0.1 --port 5187 --strictPort
```

打开 `http://127.0.0.1:5187`。

## 成长系统：功法 + 法宝

- **法宝（主动）**：可装备，**上限 10 个**，**彼此按标签羁绊**（剑/刀/仙/魔/妖/体/雷/阵/木），底部一排 10 格常驻显示。**武器由法宝决定**。多余法宝进**法宝背包**，可随时装备/卸下/替换/丢弃。
- **功法（被动）**：纯被动属性，无上限、可叠加。只增强角色，不决定武器。
- **经济**：杀怪给**灵石**（货币）；灵气球只管升级。
- **升级 → 洞府市集**（暂停）：**只卖法宝**，花灵石购买（满 10 件入背包）。可管理法宝栏与背包、**查看羁绊面板**（每个标签的档位/效果/是否激活）、刷新、继续。
- **功法 → 打怪掉落**：精英/妖王必掉、普通怪小概率掉**功法卷轴**；走过去拾取 → 打开**功法商店**，花灵石参悟功法。

## 武器（由法宝解锁/增强）

- 飞剑：常驻自动开火，`剑` 法宝增加飞剑数量
- 掌心雷：装备 `雷` 法宝后解锁，AOE 爆发
- 五行阵：装备 `阵` 法宝后解锁，环身光环 + 减速

## 代码结构（prototype-3d/src）

- `main.ts` — 主循环、市集、法宝栏/功法条 UI、自动武器、HUD、结算；顶部 `TUNING` 集中所有数值
- `upgrades.ts` — `TREASURES`(法宝) / `ARTS`(功法) 数据 + 羁绊 `SYNERGY` + `computeStats` / `weaponFlags` / `rollOffers`（纯逻辑）
- `models.ts` — low-poly 网格（剑修、傀儡怪、石碑）
- `effects.ts` — 打击/雷击/阵法/拾取特效
